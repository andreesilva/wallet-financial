<h1 align="center"> Carteira Financeira </h1>



## 💻 Sobre o projeto


📄 Carteira financeira em que os usuários possam realizar transferência de saldo. Usuários podem enviar ou receber dinheiro.

## 🛠 Tecnologias


- NodeJs
- NestJs
- TypeScript
- Prisma
- MySQL
- Swagger
- Docker
